# 🛒 E-Commerce Sales Dashboard (Tableau)

## 📌 Project Overview
This project analyzes e-commerce sales data to identify trends in sales, products, payment methods, and order status.

## 📊 Dashboard Features
- KPI Metrics: Total Sales, Total Orders, Quantity Sold
- Sales by Product
- Sales by State (Geographical Analysis)
- Payment Method Analysis
- Order Status Distribution
- Sales Trend Over Time

## 🧹 Data Cleaning
- Handled missing values
- Removed duplicates
- Standardized date formats
- Cleaned categorical columns

## 🛠 Tools Used
- Tableau
- Excel / Python (for cleaning)

## 📷 Dashboard Preview
![Dashboard](dashboard.png)


## 📈 Key Insights
- Tablets generated highest sales
- COD is the most used payment method
- Peak sales observed on Jan 16
- Majority orders are delivered successfully

## 🚀 How to Use
- Open Tableau file (.twbx)
- Explore filters (State, Payment Method, Status)
